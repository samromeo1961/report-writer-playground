# Report Writer — Playground

**▶ Live: https://samromeo1961.github.io/report-writer-playground/**

This repo hosts the **built, static playground** for the Report Writer engine — a
framework-agnostic, dependency-light **banded report engine + visual designer**
with built-in Word/Excel **template-merge**. It runs entirely in your browser; no
server, no sign-up, no data leaves the page.

## What to try
- **Design** — drag bands/elements on the canvas; edit properties; group, aggregate, sub-report.
- **Preview** — paginated, interactive preview with a PDF-style toolbar.
- **Sample reports** — load a bundled sample (invoice, job cost, dashboard, labels, statement…).
- **Export** — download the current report as **PDF**, **Word (.docx)** or **Excel (.xlsx)**.
- **Templates** — pick a starter Word/Excel template (or upload your own) with `[Field]` /
  `[#Lines]` / `[#if]` tags, fill the data, and merge & download.
- **JSON** — inspect/lint the report definition; copy it.

## Note
This repository contains only the **compiled demo** (HTML/JS/CSS). The TypeScript
source lives in a separate, private repository. The demo is rebuilt and republished
from there.
